# Currency Converter API

Современный сервис для конвертации валют с веб-интерфейсом, RESTful API и админ-панелью.

![FastAPI](https://img.shields.io/badge/FastAPI-005571?logo=fastapi)
![Python](https://img.shields.io/badge/Python-3.13-blue?logo=python)
![SQLite](https://img.shields.io/badge/SQLite-07405E?logo=sqlite)
![Bootstrap](https://img.shields.io/badge/Bootstrap-7952B3?logo=bootstrap)

## 🚀 Основные возможности

- **Конвертация валют** в реальном времени
- **История операций** для каждого пользователя
- **Админ-панель** для управления курсами валют и пользователями
- **RESTful API** с полной документацией
- **JWT аутентификация** с безопасным хранением токенов в cookies
- **Поддержка 150+ валют** (демо-режим с основными валютами)

## 🛠 Технологии

- **Backend**: FastAPI, SQLAlchemy, Pydantic
- **Database**: SQLite (для разработки), PostgreSQL (готов к миграции)
- **Frontend**: Jinja2, Bootstrap 5, Font Awesome
- **Security**: JWT, bcrypt, HTTPS-совместимые cookies
